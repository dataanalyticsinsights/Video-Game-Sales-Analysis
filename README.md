# Video-Game-Sales-Analysis

# Project Overview

This project analyzed historical video game sales data to identify trends and factors that influence commercial success in the gaming industry. Sales data up to 2016 was used to explore market behavior and support sales forecasting for 2017.

The full workflow, including data cleaning, preprocessing, analysis, and visualizations, was completed in the games_sales.ipynb notebook.

# Objective

The main goal of the analysis was to understand which game characteristics — such as genre, platform, and review scores — have the greatest impact on sales performance. The findings can help support forecasting and more data-driven marketing decisions.

# Methodology

The project included:

Data cleaning and preprocessing,
Exploratory data analysis (EDA),
Correlation analysis,
Sales trend analysis,
Data visualization,

The analysis focused on identifying relationships between game features and overall sales performance.

# Key Findings
Review Scores and Sales

Critic ratings showed a stronger relationship with sales performance than user ratings, particularly for platforms such as Wii, PS2, and PS4. Higher critic scores were generally associated with higher sales.

# Genre Performance

Action, sports, and shooter games consistently generated the highest sales revenue across multiple platforms.

# Platform Lifecycle Trends

The analysis highlighted changes in platform popularity over time. Older platforms such as PS2 reached peak popularity earlier, while PS4 was expected to remain commercially strong in 2017, reflecting the typical lifecycle of gaming consoles.
